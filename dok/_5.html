Робота з SVG-спрайтами

У шаблоні “Чертоги Фрілансера” v3.0.0 можна автоматично створювати SVG-спрайти.

Створення SVG-спрайту

Для цього слід скопіювати окремі SVG іконки у папку src/svgicons (якщо такої папки немає, потрібно її створити). Далі запускаємо команду у терміналі npm run sprite

☝ Створювати SVG-спрайт слід до виклику команди npm run dev

Після виконання команди окремі іконки SVG зберуться в єдиний файл (спрайт) simg/icons/icons.svg

Якщо набір окремих іконок у папці src/svgicons змінився, слід повторити команду npm run sprite

Використання SVG-спрайту

Кожній іконці в загальному файлі (спрайті) буде наданий унікальний ID, що складається з рядка svg-ім’я-файла-іконки. Наприклад, якщо ім’я файлу окремої іконки було list.svg, то ID буде svg-list. Отже, вивести необхідну іконку в потрібному місці можна за допомогою коду:

‹svg›
‹use› xlink:href="img/icons/icons.svg#svg-list"‹/use›
‹/svg›:
